# PacketReplay
A simple utility that reads a PCAP file and replays all packets via TCP/IP


# Windows 10 Dependencies
Boost C++ 1.83.0
https://www.boost.org/users/history/version_1_83_0.html
https://sourceforge.net/projects/boost/files/boost-binaries/1.83.0/
https://www.boost.org/LICENSE_1_0.txt

WinPcap 4.1.2 Developer's Pack
https://www.winpcap.org/devel.htm
https://www.winpcap.org/misc/copyright.htm

PcapPlusPlus September 2023 Release
https://github.com/seladb/PcapPlusPlus/releases/tag/v23.09
https://github.com/seladb/PcapPlusPlus/releases/download/v23.09/pcapplusplus-23.09-windows-vs2022-x64-release.zip
https://github.com/seladb/PcapPlusPlus/blob/master/LICENSE
